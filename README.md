# Sales Analytics Dashboard (2023 - 2024)
An interactive Excel-based sales dashboard designed to analyze and visualize performance across multiple dimensions. This project consolidates data from 7 different source files into a single, professional reporting interface.

# Key Features
Performance Tracking: Real-time monitoring of Total Sales ($974,693) and Active Customers (638).

Trend Analysis: Monthly sales tracking to identify seasonal patterns.

Categorical Insights: Deep dives into sales by Brand, Category, and Branch performance.

Payment Analysis: Comparison of revenue generated via PayPal, Credit Cards, Cash, and Bank Transfers.

Dynamic Filtering: Built-in Slicers (Month, Category, Branch) for instant data drill-down.

# Project Structure
The project is built on a robust architecture of 7 Excel files


# Tools Used
Microsoft Excel: Power Query, Power pivot, Pivot Tables, Advanced Charts, Slicers, and Data Modeling.
